Subscreasy Payment Gateway plugin is a payment gateway plugin for WooCommerce and WooCommerce Subscriptions.
It provides both single payments and recurring payments.
Currently, the plugin supports payments through iyzico, PayU, PayTR, Stripe payment service providers can be used with this plugin.
